CROUSE CONTROL

Hardware optimization utility for ARC Raiders. Forces Windows NT CPU thread prioritization and executes memory standby list purging to prevent resource starvation.

Download Version 1 (Includes Discord Audio Fix) Download Version 2 (ARC Raiders Optimization Only)

Execution Protocol

  1. Download the designated .exe binary.
  2. Execute the file prior to launching ARC Raiders.
  3. Approve the Windows User Account Control (UAC) prompt. (Administrator privileges are strictly required to interface with the NT kernel memory state).
  4. The utility will run silently. Verification of execution is indicated by the shield icon in the system tray.
  5. To terminate the background process, right-click the system tray icon and select "Exit Crouse Control".
* Note: This program only operates in your temporary memory (RAM) while you game. It makes zero permanent changes to your Windows settings or registry.
